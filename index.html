üé§ Edge TTS Voice Selection:

Edge TTS is completely free and requires no API key! Simply:

  1. Select your preferred voice from the dropdown menu
  2. Click "Save Voice" to save your selection
  3. Test the voice using the test generation feature
  4. Edge TTS supports multiple languages and natural-sounding voices

üé® Image Generation (Leonardo.ai / Pollinations AI):

Free Option - Pollinations AI (Default):

  • Completely free with no API key required
  • High-quality image generation using Stable Diffusion
  • Simply leave the Leonardo.ai toggle OFF to use this free service
  • Visit Pollinations.ai to learn more

Premium Option - Leonardo.ai:

  1. Enable the "Use Leonardo.ai" toggle in settings
  2. Visit Leonardo.ai Dashboard
  3. Sign in to your Leonardo.ai account
  4. Go to API Settings
  5. Generate a new API key and copy it
  6. Premium service offers more control and specialized models

üìÖ Late API Key:
